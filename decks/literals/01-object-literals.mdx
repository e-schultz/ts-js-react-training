# Object - Basics 

* Key / Value Pair 
* Value can be any type - including other objects, arrays, functions, etc
* Keys can be added / removed after an Object has been created
* Can access keys dynamically

  return obj[prop]
};

# Object Literals

---

## What are they?

- Key name becomes property name
- Short hand for creating objects
- Can use computed key names

---

## The Old Way

```javascript
function createCar(make, model, year) {
    return {
        make: make,
        model: model,
        year: year
    }
}
```

---

## Shorthand

```javascript
function createCar(make, model, year) {
    return {
        make,
        model,
        year,
    }
}

console.log(createCar('Toyota','Camry','2018'))
// { make: 'Toyota', model: 'Camry', year: '2018'}
```
